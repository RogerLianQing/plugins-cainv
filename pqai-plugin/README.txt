PQAI Wordpress Plugin

== Description ==

This plugin is to add PQAI search capabilities to your wordpress website.
PQAI search can be accessible from admin panel and plugin php file for public
use.

== Installation ==

1. Add your API token to the `api/search.php` file.
2. Configure the number of results you want to show on the `api/search.php` file.
3. Upload `pqai-plugin` to the `/wp-content/plugins/` directory.
   Or you can use unzip method in 'Plugins' menu at admin dashboard.
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Click on "PQAI Search" link.
