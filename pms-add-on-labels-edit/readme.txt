== Description ==

Paid Member Subscriptions - Labels Edit Add-on

**NOTE:** In order for this plugin to work, you need to have [Paid Member Subscriptions](https://wordpress.org/plugins/paid-member-subscriptions/) installed and activated.

This plugin adds a subpage to the [Paid Member Subscriptions](https://wordpress.org/plugins/paid-member-subscriptions/) menu that will let you change any string that is coming from the plugin.

Examples of what you can edit:
* button labels
* field labels
* headings
* success/error messages that the plugin is showing
* & more

== Installation ==

1. Upload and install the zip file via the built in WordPress plugin installer.
2. Activate the plugin from the "Plugins" admin panel using the "Activate" link.
